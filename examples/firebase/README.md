# With Firebas

## How to use

1. Create a Firebase project and add a new app to it.
2. Create a .env file and copy the contents of .env.example into it:

```zsh
cp .env.example .env
```

3. Set each variable on .env with your Firebase Configuration.

Install it and run

```zsh
yarn install
yarn dev
```
